Apresentação do site

O site originalmente foi um projeto que fiz logo após aprender desenvolvimento web, por isso tentei utilizar varios recursos para por em pratica o que aprendi, certamente tomar uma abortagem mais minimalista tornaria o site mais agradevel ao usuario, mas o objetivo do site era por em pratica o conhecimento recem adquirido.
Dito isto, o site originalmente possuia conexão com um banco de dados para armazenar dados das imagens, mas como o github pages não possui a possibilidade de criar um banco de dados, tive que adaptar o site para ser um site estatico, por isso ele não esta tão bom quanto deveria, especialmente na pagina do administrador.
A pagina do administrador é a pagina em que o dono do site pode modificar tudo que quiser sem precisar programar, mas como todas as modificações precisariam se comunicar com o banco de dados, as funções da pagina do administrador não funcionam.
